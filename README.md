# ToJo

This is the repository for the ToJo app.

## Getting started

To get started and run the project, please follow the instructions below:

1. Install backend dependencies

```sh
    cd backend
    npm install
```

2.  Install frontend dependencies

```sh
    cd frontend
    npm install
```

3. Insall dev dependencies

```sh
    npm install
```

4. Run the project

```sh
    npm start
```
